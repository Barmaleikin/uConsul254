Simple and naive Rust implementation of Consul 254 typewrite (autonomus mode).

Build for Ubuntu, uses iced lib.

Start typing from bottom line. use Enter to Line feed and Carriage return, or Alt+Enter for Carriage return. 
Allows to type in latin and cyrrilic top case letters, numbers and some other symbols.

Use cursor left and Right keys to move carret over line. Allowed to type three symbols might be type for same position (see below about cursor keys) over existing with coliur mixing (giver different colors combinations).

Space just skips unused space or highlights typed symbol at current position (a little fantasy featury, never existed in real device).
